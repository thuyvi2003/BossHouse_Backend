1/Clone repo về máy:

git clone <repo_url>
cd BossHouse_Backend


2/Tạo nhánh mới để phát triển tính năng hoặc sửa bug:

git checkout -b ViVLT


3/Thực hiện code và commit:

git add .
git commit -m "Implement promotion list API"


4/Đẩy nhánh lên remote:

git push origin ViVLT


5/Tạo Pull Request (PR) trên GitHub → merge vào develop

****Cách đặt tên nhánh studentCode ví dụ: ViVLT
****Cách commit tên nhánh + mô tả công việc ví dụ: "ViVLT: Edit promotion function"
